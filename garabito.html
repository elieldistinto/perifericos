Bateria CMOS

A bateria CMOS fornece energia ao chip BIOS, permitindo que ele armazene as configurações do sistema, como data e hora, e outras configurações essenciais. Ela é uma bateria de lítio, geralmente do tipo CR2032, e garante que o sistema preserve essas informações mesmo quando o computador está desligado.

Função da Pilha (Stack)

A pilha (stack) é uma estrutura de dados usada para armazenar informações temporárias, como o endereço de retorno de funções, variáveis locais e controle de fluxo do programa. Ela segue a política LIFO (Last In, First Out), onde o último elemento a ser inserido é o primeiro a ser retirado.

O que é ULA?

A ULA (Unidade Lógica e Aritmética) é um componente essencial em qualquer processador. Ela é responsável por realizar operações lógicas (como AND, OR, NOT) e aritméticas (como soma e subtração) entre os dados processados pela CPU.

Memória RAM e ROM

A memória RAM (Memória de Acesso Aleatório) é volátil, ou seja, perde seus dados quando o computador é desligado. Ela é usada para armazenar temporariamente os dados e programas em execução. Já a memória ROM (Memória Somente de Leitura) é permanente e contém dados que não podem ser alterados, como o firmware do sistema.

Overclocking

Overclocking é o processo de aumentar a velocidade de operação de um componente do computador, como o processador ou a placa gráfica, além das especificações recomendadas pelo fabricante. Isso pode melhorar o desempenho, mas também aumenta o risco de aquecimento excessivo e danos aos componentes.

Memória Cache

A memória cache é uma memória de acesso ultrarrápido, localizada entre a CPU e a RAM. Ela armazena dados e instruções frequentemente usados, permitindo que a CPU acesse essas informações mais rapidamente, melhorando o desempenho do sistema.

Barramento de Dados

O barramento de dados é um sistema de comunicação que permite a transferência de dados entre a CPU, memória e outros componentes do computador. Ele é composto por uma série de fios ou canais de comunicação que transportam informações em forma de sinais digitais.

IR (Instruction Register)

O IR (Registrador de Instrução) é um registrador na CPU que armazena a instrução atual que está sendo executada. Ele contém os dados que a unidade de controle vai decodificar e processar para realizar a operação correspondente.

Microcontroladores

Os microcontroladores são circuitos integrados que contêm uma CPU, memória e periféricos necessários para controlar sistemas eletrônicos. São utilizados em dispositivos como eletrodomésticos, automóveis, dispositivos IoT, entre outros, oferecendo soluções de baixo custo e alta eficiência energética.

Interrupções e Exceções

Interrupções são sinais enviados para a CPU para interromper o processo atual e permitir que ela execute uma tarefa mais urgente. Elas podem ser originadas por hardware ou software. Já exceções são eventos que ocorrem devido a erros durante a execução do programa, como tentativa de divisão por zero ou acesso inválido à memória, exigindo que o sistema lide com a situação para evitar falhas.
